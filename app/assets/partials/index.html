Event Cracker

Simple application that:

  1. Receives stream of events as input
  2. Maintain list of subscribers that can subscribe to the events of certain type
    1. An output channel can be configured for a given subscriber to process events that he subscribed to
  3. Maintain list of output channels that are sources of the events
